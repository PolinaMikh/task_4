Это реализация кэширующего dns-сервера.

-Для работы программы нужна библиотека dnslib(чтобы парсить dns-ответ и составлять dns-запрос).
-Программа прослушивает порт 53.
-Сервер сериализует данные в файл cache.txt, который создается при завершени программы

Сначала необходимо запустить файл 'main.py' - это dns-сервер, затем - 'server.py', чтобы отправить запрос для vk.com
